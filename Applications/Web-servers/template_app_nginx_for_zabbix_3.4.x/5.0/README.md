Installation
------------


### Nginx Configuration


 


Add the following example to your default vhost configuration file:



```
location /nginx\_status {  
    stub\_status on;  
    access\_log off;  
    allow 127.0.0.1;  
    allow ::1;  
    deny all;  
}
```
