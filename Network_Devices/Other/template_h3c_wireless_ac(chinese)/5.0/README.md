# H3C AC

## Overview

# H3C 无线控制器  
  
### 功能  
  
- CPU、内存使用率  
- 序列号  
- 流量  
- 各种认证协议在先用户数量  
- 在线AP数  
- DHCP地址池发现，地址池满报警  
- AP发现，AP用户数，AP在线时间监控，AP断线报警  
- 一张聚合图形  
  
### 宏  
  
- {$SNMP\_COMMUNITY}: SNMP v2 团体字  
- {$DHCP\_POOL\_H}: 地址池阈值  
  
### 其他  
  
- CPU、内存的硬件序号因设备型号而定，需要按照型号不同进行调整
