# SFP Mikrotik

## Overview

Template para monitoramento de interfaces SFP de equipamentos Mikrotik


 


Monitora:


 - Tx


 - RX


 - Temperatura


 


Triggers:


 - Variação de sinal de tx/x


 - Alarme de temperatura


 


Link GitHub:


<https://github.com/anderson-raber/sfp_mikrotik/blob/master/README.md>
